# Memory Card Game

A two-player memory matching card game with a 3x3 grid hosted on Nginx.

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Features

- 3x3 grid with matching pairs
- Two-player turn-based gameplay
- Score tracking
- Game state persistence

## Technologies

- React.js
- CSS
- Nginx
